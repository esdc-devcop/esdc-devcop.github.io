This form should be filled out by the Software Factory group-Groupe de manufacture de logiciels (SF-ML). The SF-ML team will create the Service Account Name and open a service ticket to the Directory Services group. The Directory Services group are responsible for granting the client account permission to our web services.

    The service account must meet the following criteria:
  • The client should NOT already have an existing prod account.
  • The client should create a non-prod account (dev) before requesting a prod account.
  • It should be created in the .NET domain.
  • The service account name must start with sv-sf.
  • The account name should match the one already created in development (ie, sv-sf-acctname-dev).
